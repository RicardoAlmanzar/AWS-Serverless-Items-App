# AWS Serverless Items App

Serverless CRUD web application built with AWS Lambda, API Gateway, DynamoDB, S3 and CloudFormation.

## Overview

This project is a full-stack cloud application where the frontend is hosted as a static website in Amazon S3 and the backend is powered by API Gateway, AWS Lambda and DynamoDB.

## Features

- View items
- Create new items
- Delete items
- Serverless backend
- Infrastructure as Code using CloudFormation

## Architecture

- **Frontend:** React + TypeScript
- **Hosting:** Amazon S3
- **API:** Amazon API Gateway
- **Compute:** AWS Lambda
- **Database:** Amazon DynamoDB
- **Provisioning:** AWS CloudFormation
